日本語

よくある質問

データはどこに保存されますか？

すべてお使いの端末内にのみ保存されます。外部サーバーへの送信や銀行連携は一切ありません。機種変更時はデータの引き継ぎ機能がないため、あらかじめご了承ください。

支払い前の通知が届きません。

iPhoneの「設定 → 通知 → SubScope」で通知が許可されているかご確認ください。また、各サブスクの編集画面で「支払い前に通知」がオンになっているか、通知タイミングをご確認ください。

「開発者への支援」とは何ですか？

個人開発を応援していただくための任意のチップ（App内課金）です。購入しても機能が増えることはなく、純粋な応援としてご利用いただけます。

解約したサブスクはどうなりますか？

削除せず「解約済み」にすると履歴として残り、「解約で浮いたお金」として集計されます。完全に消したい場合は削除してください。

FAQ

Where is my data stored?

Everything is stored only on your device. Nothing is sent to any server and there is no bank linking. Please note there is no data transfer between devices.

I’m not receiving payment reminders.

Check that notifications are allowed under Settings → Notifications → SubScope on your iPhone. Also confirm that “Notify before payment” is turned on and the lead time is set in each subscription’s edit screen.

What is “Developer Support”?

It’s an optional tip (in-app purchase) to support independent development. It unlocks no features — it is purely a way to show support.

What happens to a cancelled subscription?

Marking it as “cancelled” (instead of deleting) keeps it as history and counts toward “money saved by cancelling.” Delete it if you want to remove it entirely.
